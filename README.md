# Froxlor API

## Usage

`composer require global ortinteractive/froxlor`

Set login credentials:

`froxlor login baseurl username password`

Set config values:

```bash
froxlor config:set ssh_login YOURUSERNAME
froxlor config:set host SSH_SERVER
```
